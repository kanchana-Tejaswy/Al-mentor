## Packages
framer-motion | Smooth chat animations and page transitions
nanoid | Generating unique IDs for local storage conversations
date-fns | Formatting timestamps for chat messages
lucide-react | Clean, minimal icons
clsx | Conditional class name joining
tailwind-merge | Merging tailwind classes without conflicts

## Notes
- Uses local storage for saving conversations (no backend persistence for chat history)
- Expects `POST /api/chat` to handle the AI response generation
- Design implements a forced dark theme with glowing orange accents inspired by Apple/Linear
